Using the Return List View

The Return List view displays all of the returns that you have access to. You can use groups and lookups to narrow the list of records that display.

Your security access determines what functions are available. Contact your administrator for any access rights changes. WebViewer users may not access full functionality.

How Do I?

Open the Return List view
  • On the Navigation Bar, click Returns.

Use the Summary View

Use the Lookup to search for records
Open the Return Detail view
  • Click the return's ID to open the Return Detail view.
