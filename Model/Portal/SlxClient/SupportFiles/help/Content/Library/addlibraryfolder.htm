Adding, Renaming, or Deleting a Library Folder

For better organization, you can use Library folders to group documents together.

Only the system administrator and users with an administrator role can access these features.

How Do I?

Add a library folder
  1. On the Navigation Bar, click Library.
  2. Select the folder under which you want to add the item, and click Add Folder.
  3. In the New Folder dialog box, type a name for the new folder.

  4. Click OK.

Rename a folder
  1. On the Navigation Bar, click Library.
  2. Select the folder you want to rename and click Edit Folder Name.
  3. In the dialog box, type a new name for the folder.

  4. Click OK.

Delete a folder
  1. On the Navigation Bar, click Library.
  2. Select the folder you want to delete, and click Delete Folder.
  3. Click Yes to confirm the deletion.

    4. All folders and files within the folder will also be deleted.
