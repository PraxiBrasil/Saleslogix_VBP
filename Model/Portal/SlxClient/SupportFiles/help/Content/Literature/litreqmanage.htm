Managing Literature Requests

Use the Literature Requests list view to view, fulfill, complete or reject literature requests.

How Do I?

Open the Literature Request list view
  • On the Tools menu, click Literature Request Management.
View or hide the details pane
  1. On the Literature Requests list view, select a record.
  2. If necessary, click Detail.

The Details pane displays at the bottom of the list view with detailed information about the selected record.

  1. To hide the Details pane, click Hide Detail.
Use the Lookup to search for records

Reject literature requests

Refresh the list
  • In the Task Pane, under Literature Tasks, click Refresh.
Open the Literature Request detail view
  • On the Literature Request List view, click the Send by link to open Literature Request detail view.
