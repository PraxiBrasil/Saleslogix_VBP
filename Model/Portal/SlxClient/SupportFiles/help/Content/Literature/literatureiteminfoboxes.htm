Literature Item Information Boxes

The following table describes the default information boxes.

 

Box

Description/Action
Item Number

The document ID as assigned by your literature manager.

You are not required to enter data in the Item Number box. Use the Item Number box if your company has designated document numbers for each literature type.
Name The name of the document
Author The document's author.
Family Click the ellipsis button and select an item from the list
Cost The per unit cost for the document.
On Hand

The quantity currently in inventory.

This number decreases every time a request is fulfilled. If you do not use the On Hand box, the on hand quantity does not decrease. If you enter a zero in the On Hand box, the on hand quantity decreases to a negative number.
Reorder Qty

The minimum quantity that indicates it is time to restock

If the On Hand quantity equals or is less than the Reorder quantity, a message is displayed indicating it is time to restock.
File If the document is a file, type the path to the file in the File box.
Expires

Use the Calendar button to select a date, and then click OK.

Once the date passes, the document appears in red in Literature Manager; and can be updated as necessary.
