Fulfilling, Completing or Rejecting Literature Requests

Use the Literature Request List view Common Tasks to fulfill, complete or reject literature requests.

How Do I?

Fulfill literature requests

During fulfillment, a fulfillment record is added to the Notes/History tab for each contact and account associated with the request.

  1. On the Tools menu, click Literature Request Management.
  2. Select one or more literature requests from the list.

    3. To select more than one record, hold down [Ctrl] or [Shift], and highlight the records.

  3. To print mailing labels or addressed envelopes, select a label or envelope format from the Print Labels list.
  4. In the Task Pane, under Literature Tasks, click Fulfill.

A list of the required literature for the request will print at your designated printer, along with a copy of any cover letter that was included in the request or any labels or envelopes.

  1. Gather the required literature and prepare it for delivery. After the literature has been sent, complete the literature request.

Complete literature requests

During Completion, a completion record is added to the original literature request so that the originator knows the status of the request.

  1. On the Tools menu, click Literature Request Management.
  2. Select one or more literature requests from the list.

    3. To select more than one record, hold down [Ctrl] or [Shift], and highlight the records

  3. In the Task Pane, under Literature Tasks, click Complete.
Reject literature requests

Rejecting a literature request deletes the literature request record from the list and from all associated contacts and accounts.

  1. On the Tools menu, click Literature Request Management.
  2. Select one or more literature requests from the list.

    3. To select more than one record, hold down [Ctrl] or [Shift], and highlight the records

  3. In the Task Pane, under Literature Tasks, click Complete.
Refresh the list
  • In the Task Pane, under Literature Tasks, click Refresh.
