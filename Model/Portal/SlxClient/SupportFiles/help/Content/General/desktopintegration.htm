Installing and Using Sage SalesLogix Desktop Integration

The Sage SalesLogix Desktop Integration Module is a collection of features to enhance your Sage SalesLogix Web experience.

These features include:

  • Drag and Drop Attachments
  • Drag and Drop E-mail from Microsoft Outlook
  • Outlook Integration - Send SLX and Record to History
  • Mail Merge
  • Export to Excel
  • Drag and Drop Library files (Administrator only)

How Do I?

Install Desktop Integration Module if Internet Explorer is my browser
  1. Do one of the following:
  • On the Sage SalesLogix Web Client login screen, click Enhance SalesLogix.
  • In Options, click the General/Search tab and click Enhance SalesLogix.
  • If prompted to install Sage SalesLogix Desktop Integration and you want to use this feature, click Yes.
  1. Click Run.
  2. Close Microsoft Outlook and any open browser windows.
  3. On the Internet Explorer - Security Warning window, click Run.

If prompted that Microsoft Outlook or Internet Explorer is in use, close each program, and click Retry.

  1. On the Welcome screen, click Next.
  2. On the License Agreement screen, read the information, accept the agreement, and then click Next.
  3. Click Install.
  4. Click Finish.

    5. The Desktop Manager opens, which allows you to set your Desktop Integration options.

    For more information about Desktop Manager, refer to the Desktop Manager Online help system.

    To open the Sage SalesLogix Desktop Manager manually, in the System Tray section of the Windows Taskbar, double-click the Sage SalesLogix Desktop Manager icon.
Install Desktop Integration Module if Firefox is my browser
  1. Do one of the following:
  • On the Sage SalesLogix Web Client login screen, click Enhance SalesLogix.
  • In Options, click the General/Search tab and click Enhance SalesLogix.
  • If prompted to install Sage SalesLogix Desktop Integration and you want to use this feature, click Yes.
  1. Click Run.
  2. Click Save File.
  3. Click Run.
  4. In the Downloads dialog box, double click SLXDesktopIntegrationSetup.exe.
  5. In the Open Executable File dialog box, click OK.
  6. Click Run.
  7. Close Firefox and any open browser windows.

    If prompted that Microsoft Outlook or Firefox is in use, close each program, and click Retry.

  1. On the Welcome screen, click Next.
  2. On the License Agreement screen, read the information, accept the agreement, and then click Next.
  3. Click Install.
  4. Click Finish.

    5. The Desktop Manager opens, which allows you to set your Desktop Integration options.

    For more information about Desktop Manager, refer to the Desktop Manager Online help system.

    To open the Sage SalesLogix Desktop Manager manually, in the System Tray section of the Windows Taskbar, double-click the Sage SalesLogix Desktop Manager icon.
