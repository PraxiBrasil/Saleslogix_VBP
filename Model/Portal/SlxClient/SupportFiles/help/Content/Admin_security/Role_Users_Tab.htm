Users Tab

Use the Users tab to assign one or more users to a role or to remove a user from a role. When a user is assigned to a role, the user has access to the actions defined in that role.

How Do I?

Assign a user to a role

  1. Open the Role Detail view.
  2. Click the Users tab.
  3. Click Add.
  4. Use the Lookup to find and select the user to which you want to assign the role.
  5. Click Save.

Remove a user from a role

  1. Open the Role Detail view.
  2. Click the Users tab.
  3. Click the Remove link for the user you want to remove from the role.
  4. Click OK to confirm.
  5. Click Save.
