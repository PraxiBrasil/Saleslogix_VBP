Deleting Calculated Fields

 System Administrator Only

You can only open the Manage Calculated Fields dialog box if you are logged on as an administrator.

To delete a calculated field

  1. Open Query Builder, and then click the Calculations button.

  2. In the Manage Calculated Fields dialog box, select a calculated field from the grid.

  3. Click Delete.

  4. Click OK to confirm deletion.

  5. Click Close.
