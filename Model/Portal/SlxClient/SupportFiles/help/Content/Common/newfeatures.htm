What's New In This Release

Sage SalesLogix Web Client contains the following updates:

Version 7.5.3

  • Added the ability to manage:
  • Competitors - add, edit, or delete competitors that can be associated to an opportunity.
  • Departments - add, edit, or delete departments that can be used as a specialized team.
  • Lead Sources - add, edit, or delete lead sources that identify how a contact or lead was created.
  • Literature Items - add, edit, or delete literature items available for literature requests.
  • Pick Lists - add, edit, or delete pick lists to help users improve data consistency.
  • Products and Packages - add, edit, or delete products and product packages (a collection of products) that can be associated to an opportunity or sales order.
  • Teams - create and manage teams that can be used to define account ownership and access.
  • Users - add users, manage profiles, and set security.
  • Roles - manage a user's access to Web Client features.
  • Manage Library folders and documents.
  • Fulfill, complete, or reject literature requests.
  • Merge contacts or accounts.
  • Desktop Integration for:
  • Drag and Drop Attachments
  • Drag and Drop E-mail from Microsoft Outlook
  • Outlook Integration - Send SLX and Record to History
  • Mail Merge
  • Export to Excel
  • Drag and Drop Library files (Administrator only)
  • Dashboards - add, edit, and manage data with charts and Top 10 lists.

Version 7.5.2

  • Sales Orders now have a list view and detail view.
  • The Sales Orders Detail view contains a Sales Orders Snapshot with financial information that can be forwarded in an e-mail message.
  • When a Sales Order address is changed, you are now asked if you would like to update matching contact addresses.
  • Activity Reminders have been updated to allow you to complete, reschedule and delete multiple records at once and snooze and dismiss alarms for multiple activities at once.
  • You can now print a detail report from some detail views.
  • The Calendar and Activity views now identify activities that have attachments.
  • Use the Default Lookup Layout option to select the group with the layout you want to use for lookups on each main view.
  • If you navigate to a detail view from another related detail view, the Lookup Results tab will contain records with a similar relationship. For example, other contacts related to the same account.

Version 7.5

  • Activities enhancements including attachments for activities and the ability to associate an activity to a lead.
  • Intellisync for SalesLogix has been updated to use the Intellisync version 7.0 engine.
