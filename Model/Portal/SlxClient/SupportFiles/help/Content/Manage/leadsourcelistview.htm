Using the Lead Sources List View

Only the system administrator and users with an administrator role can access these features.

The Lead Sources List view displays all of the lead sources that you have access to. You can use groups and lookups to narrow the list of records that display.

How Do I?

Open the Lead Sources List View
  • On the Navigation Bar, click Lead Sources.

Use the Lookup to search for records

Open the Lead Source Detail View
  • Click the lead source name to open the Lead Source Detail view.
