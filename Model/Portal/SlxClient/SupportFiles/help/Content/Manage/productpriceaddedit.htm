Adding or Editing Product Prices

Only the system administrator and users with an administrator role can access these features.

You can add or edit product price information.

How Do I?

To add

  1. Do one of the following:

  • Open the Product Detail view and click on the Price Levels tab.
  • Open the Insert Product view.
  1. Click the Add Price button.
  1. Add the following information:
  • Description - Enter a price description.
  • Price - Enter or price for the product.
  • Default -Identify the default price for the product. You can choose only one default price. If you select more than one, the first price will no longer be selected.

  1. Click OK.

To edit

  1. Do one of the following:

  • Open the Product Detail view and click on the Price Levels tab.
  • Open the Insert Product view.
  1. Click Edit next to the price you want to edit.
  2. Edit any of the following information:
  • Description - Edit the price description.
  • Price - Edit a price for the product.
  • Default - Identify the default price for the product. You can choose only one default price. If you select more than one, the first price will no longer be selected.
  1. Click OK.
