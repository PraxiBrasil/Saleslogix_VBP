Product Packages Tab

Only the system administrator and users with an administrator role can access these features.

You can use the Product Packages tab view and manage product packages associated with a product.

How Do I?

Add a Product Package

Edit a Product Package

Delete a Product Package
  • Click Delete next to the package you want to edit.

The Product Packages tab displays the following information:

  • Name - The product package name.
  • Description - Description for the product package.
  • Status - The status of the package. Click the ellipsis button and select an item from the list.
