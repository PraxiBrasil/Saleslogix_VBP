Members Tab

The Members tab displays the users, departments, and teams within a team. Use this tab to manage team members.

To manage

  1. In the Team Detail view, click the Members tab.
  2. Select the Automatically add user's manager to team check box if you want to add a user's manager when adding team members.
  3. Do one of the following:
  • Click Add to add a team member. Use the Lookup to find and select the team member you want to add.
  • Click the Remove link for the team member you want to remove from the team.
  • Click the Security Profile link to change the team member's security profile within the team.
  • In the Common Tasks pane, click Replace on All Teams to replace the selected team with another user, department, or team, on all teams.
  1. Click Save.
