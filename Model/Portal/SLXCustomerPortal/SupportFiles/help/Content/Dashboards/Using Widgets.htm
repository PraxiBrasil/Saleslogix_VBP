Using Widgets

Sage SalesLogix provides several out-of-the-box widgets. You can add, edit, and remove widgets from the dashboard, and you can rearrange widgets on the page with drag-and-drop.

Using pre-defined filters, you can display metrics for your data across ranges or from distinct values and display this data in one of the charting widgets. For example, you can configure a column chart widget to display all the opportunities closed and won, by owner and the total amount of the sale, or a pie chart depicting sales won by a team.

However, when using distinct filters all charting widgets, except the pie chart, are limited to 10 data points at a time; the first ten in the selection. The pie chart widget allows you to select a +1 option which depicts the balance (over the 5 or 10 selected) rolled into an Other category. When using range filters, a data point for each defined data range is returned. This can result in a chart displaying far more than 10 slices or columns, but this setting can be edited by your system administrator.

How Do I?

Add a widget
  1. Right-click the My Dashboard tab, and click Add Content.
  2. On the Add New Content dialog box, click the Add button for the widget you want to add.
    For example, to add a pie chart widget, click Add under Pie Chart.
    The blank widget appears on the dashboard and the widget editor opens.
Edit an existing widget
  1. On the toolbar for the widget, click Edit.
  2. Make the changes necessary.
  3. Click OK.
    The widget displays the changes.
Configure a charting widget
  1. On the Edit dialog box, configure the settings that are common to all charting widgets:
    • Entity - The entity from which you want to pull the data. For example, Opportunities. All entities with list views are available to view in charts and lists.
    • Group - The group for which you want to view data. For example, All Opportunities. All groups available to a user are available to use in charts and lists.
    • Dimension - The data you want to view. For example, Probability. All current Range and Distinct filters are available as dimensions.

    		 The available dimensions are dependent on the available entity filters and other configuration settings. See your system administrator for more information.
    • Metric - The way you want to calculate the data. For example, Total Actual Amount. All entities with list views have an associated count metric, for example, CountLeads and CountAccounts. There are additional metrics for Opportunities and History.

     

  1. Configure the settings specific to each widget:
    • Bar Chart: Type a title, a caption, labels for the X- and Y-axes, and then select Yes or No to display names for each bar.
    • Column Chart: Type a title, a caption, labels for the X- and Y-axes, a goal, and then select Yes or No to display names for each column.
    • Funnel: Type a title for the chart, and then select Yes or No to display names for each segment.
    • Line Chart: Type a title, a caption, labels for the X- and Y-axes, and then select Yes or No to display names for each data point.
    • Pie Chart: Type a title, select how many slices to display, select Yes or No to add a slice for "Other", and choose whether to display a legend and names for each slice.
  2. Click OK.
    The widget displays the data as configured.
Configure the Group List widget
  1. On the Edit dialog box, select an Entity and a Group from the drop-down lists. For example, Opportunities, and Closed Won.
  2. Type the Number of Records you want in the list. For example, 10, 25, or 50.
  3. Type the number of records you want visible in the Visible Records box; 10, 25 or 50.
    For example, if you typed 50 records in step 2, but only want to see ten at a time in the list, type 10. The widget resizes to accommodate the number of visible records.
  4. Click OK.
    The widget displays the list.

    		5.

    You can use any number between 1 and the maximum number of records configured to display. The default maximum is 50.
Configure the Links List widget
  1. On the Edit dialog box, type a title for the widget.
  2. Click Add Link.
  3. Double-click the Icon URL box, and type the path to the icon you want to display for the link. For example, /google.jpg.

    		4.

    This URL must be a relative path to an image on the Web server.

     

  4. Double-click the Link URL box, and type the URL for the link you want to display. For example, http://localhost/SlxClient/Account.aspx.

    5. This path can be a relative path within the SLXClient portal, or an absolute link to an external site. The URL for an external site must include the http:// designator. For example, to access www.google.com the URL must read http://www.google.com. Otherwise the browser will search for http://<SLXClient site>/www.google.com.

     

  5. Double-click the Link Text box, and type the text you want to display as the link. For example, Google Search Page.
  6. Double-click the Target box, and type an identifier for the target. For example, if you want the link to open in a new browser window, type New Window. When you click the link, a new browser window/tab opens.

    		7.

    If you leave this field blank, the link opens in the active browser window/tab when clicked. If several links have the same target designation, each link replaces the other as you click them.

     

  7. Click Add Link to add another link, or click OK to close the editor.
    When you click OK, the widget refreshes to display the links.
Refresh a widget
  • On the widget toolbar, click Refresh.
    The widget displays any changes in the data.
Remove a widget
  • On the widget toolbar, click Close.
    The widget is permanently removed from the dashboard.
