Introducing Widgets

Widgets are plug-ins you can add to a dashboard. There are several types of widgets you can use to display your data. Move your mouse over the chart in a widget to display the dimension and metric selected for that widget.

  • Dimensions are the data associated with an entity you want to view. Tell me more...

For example, for the Account entity, you can view the following dimensions:

  • Account Manager
  • City
  • Industry
  • Owner
  • SIC
  • State
  • Status
  • SubType
  • Type
  • Metrics are the way you calculate the data you want to view. Tell me more...

For example, for the Account entity you can use the following metrics:

  • Average Days Since Last Activity
  • Average Revenue
  • Average Time as Customer
  • Count Accounts
  • Total Revenue

Click the link at the bottom of each widget to view more detailed information for that data. For example, if you configured the Group List widget to display All Open Opportunities, the View List link opens the Opportunities List View. Click Welcome in the Navigation Bar to return to the dashboard.

There are several types of widgets:

Charting Widgets

  • Bar - Displays the data in horizontal bars. Tell me more... For example, if you select the data for Average Order Size by Owner, the teams are displayed on the Y-axis and the number of orders on the X-axis. Each bar extends horizontally to meet the number on the X-axis that most closely matches the total.
  • Column - Displays the data in vertical columns. Tell me more... For example, if you select the data for Average Order Size by Owner, the teams are displayed on the X-axis and the number of orders on the Y-axis. Each bar extends vertically to meet the number on the Y-axis that most closely matches the total. You can add a goal to the configuration which displays as a horizontal line across the chart, depicting where the columns are in relation to that goal.
  • Funnel - Displays the data in progressive proportions in a color-coded funnel. Tell me more... For example, you can use the pipeline to represent stages in the sales process or a campaign.
  • Line - Displays the data as a series of data points connected by a line. Tell me more... For example, if you select the data for Active Marketing Campaigns by Region, the teams are displayed on the X-axis, the number of campaigns on the Y-axis and the counts are data points on the chart.
  • Pie - Displays the data in a pie separated by color with a legend of accompanying data. Tell me more... For example, if you select Accounts by Region with five slices, the chart displays a five-colored pie with a legend displaying Owners (teams) and the total number of accounts per team. You can configure the pie chart to display five or ten slices, and choose whether to display an Other slice that includes all the data not shown.

List Widgets

  • Group - Displays the group information specified. Tell me more... For example, the top ten opportunities closed won. You can sort the columns in the list and navigate to the account or entity for more detailed information with the View Group link.

    		•

    The View Group link is not available for History

  • Links - Displays links you specify. Tell me more... For example, add links you use frequently within Sage SalesLogix or on the Internet.
  • Recently Viewed - Displays recently viewed details as links. Tell me more... For example, if you viewed the details for an account and contact, links to those detail views display in the list.
